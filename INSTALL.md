1. **Clone the Repository**

   ```bash
   git clone https://github.com/JuanVilla424/teamoon.git
   ```

2. **Navigate to the Project Directory**

   ```bash
   cd teamoon
   ```

3. **Build**

   ```bash
   make build
   ```

4. **Install** (copies binary to `/usr/local/bin`)

   ```bash
   make install
   ```

### Prerequisites

- Go 1.24+
- Git
- `gh` CLI (optional, for GitHub PR features)
